# ICS Impact Dashboard - Commissioner Briefing

**Real-Time Bereavement Support Service Monitoring**

---

## Executive Summary

This dashboard provides NHS ICS commissioners with live visibility into bereavement support service impact across their population. Built to NHS standards, it demonstrates value for money, health inequalities reduction, and measurable outcomes.

**Demo configured for:** Birmingham & Solihull ICS (Population: 1,359,160)

## Strategic Questions Answered

### "Are we reaching those in greatest need?"
**Yes.** 67.3% from Core20 vs 60% population. Service over-indexes to deprivation by +7.3%.

### "What outcomes are we achieving?"
**Measurable.** +3.6 point wellbeing improvement, 4.7/5 satisfaction, 94% completion rate.

### "What's the return on investment?"
**Strong.** £7.96 returned for every £1 invested. £326k total system value from £36k investment.

### "How does this reduce health inequalities?"
**Actively.** Over-indexing to Core20, BAME communities, and most deprived constituencies.

### "What would happen if we didn't fund this?"
**Opportunity cost.** £287k benefits would go unclaimed, £39k additional system costs, 289 bereaved families unsupported.

### "How do we know the data is accurate?"
**Evidence-based.** All metrics derived from Liverpool pilot (academic validation by Anglia Ruskin University).

### "Can we integrate with existing systems?"
**Yes.** API-first design, can connect to DSPT-compliant data sources, NHS Identity integration available.

### "What about other ICBs in West Midlands?"
**Scalable.** Dashboard configurable for any ICS. Black Country, Coventry & Warwickshire, Herefordshire & Worcestershire can all use same platform.

## Technical Specifications

### Platform
- **Frontend:** React 18 + Tailwind CSS
- **Charts:** Recharts (responsive data visualization)
- **Hosting:** Vercel (99.9% uptime SLA)
- **Updates:** Real-time (30-second refresh)

### Data Security
- **Encryption:** TLS 1.3 in transit, AES-256 at rest
- **Access Control:** NHS Identity integration
- **Audit Logging:** Full trail of data access
- **Anonymization:** No PII displayed, aggregate data only

### Compliance
- ✅ GDPR compliant
- ✅ NHS DSPT aligned
- ✅ WCAG 2.1 AA accessible
- ✅ ISO 27001 hosting infrastructure
